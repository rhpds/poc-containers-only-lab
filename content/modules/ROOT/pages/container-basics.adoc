= Container Basics

== Introduction

This section demonstrates basic container operations using the RHEL shell environment.

== Working with Files

Create a test file:

[source,bash]
----
echo "Hello from container" > /tmp/test.txt
cat /tmp/test.txt
----

== Package Management

Check available packages:

[source,bash]
----
dnf list installed | head -20
----

Note: Additional package installation may require network access defined in `firewall.yaml`.

== Python Examples

Create a simple Python script:

[source,bash]
----
cat > /tmp/hello.py << 'PYEOF'
#!/usr/bin/env python3
import platform
print(f"Python {platform.python_version()}")
print(f"Running on {platform.system()} {platform.release()}")
PYEOF

chmod +x /tmp/hello.py
/tmp/hello.py
----
